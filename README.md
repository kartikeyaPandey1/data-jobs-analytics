# 🌍 Global Data Jobs Analytics Dashboard

## 📊 Overview

This project presents an interactive Power BI dashboard analyzing global trends in data-related jobs, including job demand, salary insights, and role distribution.

The objective is to transform raw data into meaningful insights that help understand industry trends and support better decision-making.

---

## 🛠 Tools Used

* Power BI (Data Visualization & Dashboard)
* CSV Dataset (External Source)

---

## 📈 Key Insights

* Data Engineer roles show the highest job demand among data-related positions
* The median yearly salary is approximately **$113K**, indicating strong earning potential
* Job opportunities fluctuate across months, showing time-based trends
* Salaries vary significantly across countries, highlighting geographic differences
